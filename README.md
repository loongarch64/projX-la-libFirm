# projX-la-libFirm

## 项目名称

libFirm编译器LoongArch架构移植。

## 支持单位

龙芯中科技术股份有限公司

## 项目描述

LibFirm是一个学术研究型的编译器。LIbFirm是一个库，它提供基于图结构的中间表示、优化和代码生成器。目前支持x86、arm64、mips、risc-v等架构。项目目标是将libFirm移植到LoongArcgh平台。
